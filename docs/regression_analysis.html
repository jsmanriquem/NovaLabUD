Módulo: regression_analysis.py#

Este es el módulo que se encarga de hacer el análisis de los datos, para ello utiliza diferentes métodos como lo son: regresión lineal, regresión polinómica e interpolación.

class regression_analysis.RegressionAnalysis(data_ops)[fuente]#

Bases: object

Clase para realizar análisis de regresión.

Esta clase proporciona métodos para realizar análisis de regresión lineal y polinómica, así como interpolación de datos. Las métricas de desempeño se pueden calcular y mostrar en una interfaz gráfica.

data_ops#

Objeto que contiene los datos necesarios para el análisis de regresión.

Type:

DataOperations

calculate_metrics(y_true, y_pred)[fuente]#

Calcula las métricas de regresión.

Parámetros:

  • y_true (array-like) – Valores reales.

  • y_pred (array-like) – Valores predichos por el modelo.

Devuelve:

(R², MAE, MSE)

Tipo del valor devuelto:

tuple

Advertencia

Asegúrate de que las dimensiones de y_true y y_pred coincidan.

interpolation(var_x, var_y)[fuente]#

Realiza la interpolación lineal de los datos.

Parámetros:

  • var_x (str) – Nombre de la variable independiente.

  • var_y (str) – Nombre de la variable dependiente.

Advertencia

Si data_ops.data es None o las variables no están seleccionadas, se mostrará una advertencia.

linear_regression(var_x, var_y)[fuente]#

Realiza una regresión lineal.

Parámetros:

  • var_x (str) – Nombre de la variable independiente.

  • var_y (str) – Nombre de la variable dependiente.

Advertencia

Si data_ops.data es None o las variables no están seleccionadas, se mostrará una advertencia.

polynomial_regression(var_x, var_y)[fuente]#

Realiza una regresión polinómica.

Parámetros:

  • var_x (str) – Nombre de la variable independiente.

  • var_y (str) – Nombre de la variable dependiente.

Advertencia

Si data_ops.data es None o las variables no están seleccionadas, se mostrará una advertencia. Si el grado del polinomio es None, no se realizará la regresión.

show_metrics(r2, mae, mse)[fuente]#

Muestra las métricas de regresión en una ventana de Tkinter.

Parámetros:

  • r2 (float) – Coeficiente de determinación R².

  • mae (float) – Error absoluto medio.

  • mse (float) – Error cuadrático medio.
